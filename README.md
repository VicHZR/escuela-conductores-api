# Escuela Conductores API

API REST desarrollada con Spring Boot 4 para la gestión de escuelas de conductores.

## Características
- CRUD de Departamento
- Búsqueda estándar y búsqueda rápida
- Paginación en base de datos
- Paginación en memoria con DTO
- Eliminación lógica
- Seguridad con Basic Authentication
- Navegabilidad REST con HATEOAS
- Documentación con Swagger (OpenAPI)

## Tecnologías
- Java 21
- Spring Boot 4
- Spring Data JPA
- Spring Security
- Spring HATEOAS
- PostgreSQL
- Maven
